# GUI Chat App

A browser-based real-time chat interface built with Node.js, Express, and Socket.IO. Multiple browser tabs or windows can connect to the same server and chat together.

## Requirements

- Node.js installed
- npm installed

## How to Run

1. Open a terminal and go to the GUI folder:

```bash
cd gui
```

2. Install dependencies:

```bash
npm install
```

3. Start the GUI server:

```bash
npm start
```

4. Open this address in your browser:

```text
http://localhost:3005
```

5. Open the same address in more browser tabs or windows to chat between multiple users.

## Features

- Responsive layout for desktop and mobile screens
- Live connected-user list
- Editable display name
- Connection status indicator
- Auto-growing message box
- Multi-line messages with `Shift + Enter`
